Суть скрипта:

Этот скрипт устанавливает или удаляет пакеты из списка:

    Читает файл со списком пакетов - построчно считывает имена пакетов

    Устанавливает пакеты - если команда install, запускает apt install для каждого пакета

    Удаляет пакеты - если команда uninstall, запускает apt remove для каждого пакета

Как использовать:

go run script.go packages.txt install    # установить все пакеты из файла
go run script.go packages.txt uninstall  # удалить все пакеты из файла

Пример файла packages.txt:

nginx
git
curl
wget

Скрипт автоматизирует массовую установку/удаление пакетов, что полезно для настройки серверов или рабочих окружений.
