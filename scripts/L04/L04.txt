Написать скрипт который автоматически создаёт нового пользователя в Linux-системе на основе переданных данных, генерирует логин по определённым правилам и выводит информацию о созданном пользователе.

Условие задачи
Цель:
Написать скрипт для автоматического создания пользователя в Linux с заданными параметрами.
Входные данные (аргументы):
$1 — имя пользователя (например, "Ivan").
$2 — фамилия пользователя (например, "Petrov").
$3 — название группы, к которой будет принадлежать пользователь (например, "developers").

Логика работы:
Генерация логина:

Берёт первую букву имени (в нижнем регистре).
Прибавляет фамилию в нижнем регистре.
Пример: "Ivan Petrov" → логин "ipetrov".

Создание пользователя:
Генерирует случайный UID ($RANDOM).
Создаёт пользователя с помощью useradd:


sudo useradd -m $login -u $user_id -g $group_name -c "$input_name $input_surname"

-m — создаёт домашнюю директорию (/home/$login).
-u — устанавливает случайный UID.
-g — назначает группу.
-c — добавляет комментарий (полное имя).

Вывод информации о пользователе:
Выводит в консоль:
Логин
Используемую оболочку (/bin/bash, /bin/sh и т. д.)
Домашнюю директорию

Группы, в которых состоит пользователь
UID
Пример использования:

./create_user.sh "Ivan" "Petrov" "developers"
Вывод:

Login: ipetrov  
Shell: /bin/bash  
Home dir: /home/ipetrov  
Groups: ipetrov developers  
UID: 54321
