Суть этого скрипта - автоматическое создание пользователя Linux с предварительно настроенным SSH-доступом.
Основная цель:

Автоматизировать процесс создания новых пользователей в системе с готовой SSH-настройкой.
Конкретные задачи:

    Создание учетной записи
     - Логин формируется автоматически (первая буква имени + фамилия)
     - Пример: Иван Петров → логин ipetrov

    Настройка SSH-доступа
    - Создается папка .ssh
    - Добавляется authorized_keys с публичным ключом
    -Это позволяет пользователю сразу войти по SSH без пароля (только по ключу)

    Безопасность
    - Правильные права доступа для SSH файлов
    - Пользователь создается в своей группе
