Суть скрипта:

Этот скрипт анализирует логи SSH и создает JSON-отчет:

    Собирает статистику SSH за указанное количество часов

    Успешные авторизации - логины с "Accepted"

    Неуспешные попытки - логины с "Invalid user"

    Извлекает данные: IP, логин, дата

    Сохраняет в JSON структурированный отчет

Что делает:

    journalctl -u sshd - получает логи SSH службы

    grep "Accepted" - успешные входы

    grep "Invalid user" - неудачные попытки

    awk извлекает конкретные поля (IP, логин, дата)

    Формирует JSON с количеством и деталями попыток

Использование:

go run script.go 24  # логи за последние 24 часа

Скрипт полезен для мониторинга безопасности SSH и анализа попыток входа.
