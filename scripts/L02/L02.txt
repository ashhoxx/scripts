Напишите скрипт на Bash, который выполняет следующие действия:

Инициализация строки:

Скрипт должен начинаться с некоторой строки (INIT_STRING), содержащей набор символов. Например, INIT_STRING=1234567890.

Проверка переменной TEST_COUNTER:

Если переменная TEST_COUNTER равна нулю, скрипт должен начать бесконечный цикл добавления чисел к строке INIT_STRING. В каждом цикле число должно увеличиваться, начиная с 1, а результат (строку INIT_STRING) нужно выводить, с задержкой между итерациями в 0.1 секунды.
Скрипт должен завершаться только при внешнем прерывании.

Поведением при ненулевом значении TEST_COUNTER:
Если TEST_COUNTER > 0, скрипт выполняет другой цикл (for), добавляя к строке INIT_STRING числа от 1 до значения TEST_COUNTER.
Для каждого значения задержка между итерациями должна быть рассчитана как TEST_COUNTER * 0.1 секунд.
Задержка вычисляется с использованием точной арифметики (bc).
